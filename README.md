# macOS Hardening

Lightweight macOS security auditing and hardening assessment script for macOS.

## Features

- Firewall checks
- SIP / Secure Boot validation
- FileVault verification
- Sharing & remote access audit
- Privacy configuration checks
- System service inspection

## Usage

```bash
chmod +x hardeningMAC.sh
./hardeningMAC.sh
```

## Requirements

- macOS
- Administrator privileges (sudo)
